## SOURCE 1: Financial Econometrics (311-330)
**Citation:** Campbell, Lo, MacKinlay (1997) "The Econometrics of Financial Markets" Princeton
**Citation:** Hamilton (1994) "Time Series Analysis" Princeton

### Formula 311: OLS Estimator
```
β̂ = (X'X)^{-1}X'y
```
**Edge:** Ordinary least squares coefficient estimation

### Formula 312: OLS Standard Errors
```
SE(β̂) = √[s² × (X'X)^{-1}]
s² = e'e / (n - k)
```
**Edge:** Statistical significance testing

### Formula 313: Heteroskedasticity-Robust SE (White)
```
V(β̂) = (X'X)^{-1} X'ΩX (X'X)^{-1}
Ω = diag(e_i²)
```
**Citation:** White (1980) Econometrica
**Edge:** Robust to non-constant variance

### Formula 314: Newey-West HAC SE
```
V(β̂) = (X'X)^{-1} Ŝ (X'X)^{-1}
Ŝ = Γ̂_0 + Σ_{j=1}^{m} w_j(Γ̂_j + Γ̂_j')
```
**Citation:** Newey & West (1987) Econometrica
**Edge:** Robust to autocorrelation + heteroskedasticity

### Formula 315: Fama-MacBeth Regression
```
Step 1: β̂_i = cross-sectional regression each period
Step 2: λ̂ = (1/T) Σ_t λ̂_t, SE(λ̂) = σ(λ̂_t)/√T
```
**Citation:** Fama & MacBeth (1973) J. Political Economy
**Edge:** Factor risk premium estimation

### Formula 316: GMM Estimator
```
θ̂ = argmin g(θ)'W g(θ)
g(θ) = (1/T) Σ_t m(x_t, θ)
```
**Citation:** Hansen (1982) Econometrica
**Edge:** General method of moments

### Formula 317: J-Test (GMM Overidentification)
```
J = T × g(θ̂)'Ŵ g(θ̂) ~ χ²(m-k)
```
**Edge:** Test for model specification

### Formula 318: Augmented Dickey-Fuller Test
```
Δy_t = α + βt + γy_{t-1} + Σ δ_i Δy_{t-i} + ε_t
H_0: γ = 0 (unit root)
```
**Citation:** Dickey & Fuller (1979) JASA
**Edge:** Test for stationarity

### Formula 319: Phillips-Perron Test
```
Z_t = t_γ × √(γ̂_0/f̂_0) - T(f̂_0 - γ̂_0)/(2f̂_0^{1/2}s)
```
**Citation:** Phillips & Perron (1988) Biometrika
**Edge:** Non-parametric unit root test

### Formula 320: Johansen Cointegration Test
```
Trace: λ_trace = -T Σ_{i=r+1}^{n} ln(1-λ̂_i)
Max-eigen: λ_max = -T ln(1-λ̂_{r+1})
```
**Citation:** Johansen (1991) Econometrica
**Edge:** Multivariate cointegration testing

### Formula 321: Error Correction Model (ECM)
```
Δy_t = α(y_{t-1} - βx_{t-1}) + Σ γ_i Δy_{t-i} + Σ δ_i Δx_{t-i} + ε_t
```
**Citation:** Engle & Granger (1987) Econometrica
**Edge:** Short-run + long-run dynamics

### Formula 322: Vector Autoregression (VAR)
```
Y_t = c + A_1 Y_{t-1} + ... + A_p Y_{t-p} + ε_t
```
**Citation:** Sims (1980) Econometrica
**Edge:** Multivariate dynamics

### Formula 323: Granger Causality Test
```
F = [(RSS_r - RSS_u)/q] / [RSS_u/(T-2p-1)]
```
**Citation:** Granger (1969) Econometrica
**Edge:** Predictive causality

### Formula 324: Impulse Response Function
```
IRF_h = ∂Y_{t+h}/∂ε_t = Φ_h
```
**Edge:** Dynamic response to shocks

### Formula 325: Forecast Error Variance Decomposition
```
FEVD_{ij}(h) = Σ_{k=0}^{h-1} (e'_i Φ_k Σ e_j)² / MSE_i(h)
```
**Edge:** Contribution of shock j to variance of i

### Formula 326: Structural VAR (Cholesky)
```
Bε_t = u_t, B = lower triangular
```
**Edge:** Identified structural shocks

### Formula 327: Information Criteria
```
AIC = -2ln(L)/T + 2k/T
BIC = -2ln(L)/T + k×ln(T)/T
```
**Citation:** Akaike (1974), Schwarz (1978)
**Edge:** Model selection

### Formula 328: Likelihood Ratio Test
```
LR = 2(ln L_u - ln L_r) ~ χ²(q)
```
**Edge:** Nested model comparison

### Formula 329: Wald Test
```
W = (Rβ̂ - r)'[R(X'X)^{-1}R's²]^{-1}(Rβ̂ - r) ~ χ²(q)
```
**Edge:** Linear hypothesis testing

### Formula 330: Hausman Test
```
H = (β̂_FE - β̂_RE)'[V(β̂_FE) - V(β̂_RE)]^{-1}(β̂_FE - β̂_RE) ~ χ²(k)
```
**Citation:** Hausman (1978) Econometrica
**Edge:** Fixed vs random effects

## SOURCE 2: Time Series Models (331-350)
**Citation:** Box, Jenkins, Reinsel (2015) "Time Series Analysis" Wiley

### Formula 331: AR(p) Process
```
y_t = c + φ_1 y_{t-1} + ... + φ_p y_{t-p} + ε_t
```
**Edge:** Autoregressive dynamics

### Formula 332: MA(q) Process
```
y_t = μ + ε_t + θ_1 ε_{t-1} + ... + θ_q ε_{t-q}
```
**Edge:** Moving average of shocks

### Formula 333: ARMA(p,q) Process
```
φ(L)y_t = θ(L)ε_t
φ(L) = 1 - φ_1L - ... - φ_pL^p
```
**Edge:** Combined AR + MA

### Formula 334: ARIMA(p,d,q) Process
```
φ(L)(1-L)^d y_t = θ(L)ε_t
```
**Edge:** Integrated (non-stationary) series

### Formula 335: SARIMA (Seasonal)
```
φ(L)Φ(L^s)(1-L)^d(1-L^s)^D y_t = θ(L)Θ(L^s)ε_t
```
**Edge:** Seasonal patterns

### Formula 336: Exponential Smoothing (Simple)
```
ŷ_{t+1} = αy_t + (1-α)ŷ_t
```
**Edge:** Weighted average forecast

### Formula 337: Holt-Winters (Additive)
```
Level: l_t = α(y_t - s_{t-m}) + (1-α)(l_{t-1} + b_{t-1})
Trend: b_t = β(l_t - l_{t-1}) + (1-β)b_{t-1}
Season: s_t = γ(y_t - l_t) + (1-γ)s_{t-m}
```
**Edge:** Trend + seasonality

### Formula 338: State Space Model
```
State: x_t = F x_{t-1} + w_t
Obs: y_t = H x_t + v_t
```
**Edge:** General dynamic linear model

### Formula 339: GARCH(1,1) Forecast
```
E[σ²_{t+h}] = ω/(1-α-β) + (α+β)^{h-1}(σ²_t - ω/(1-α-β))
```
**Edge:** Volatility mean reversion

### Formula 340: GJR-GARCH (Asymmetric)
```
σ²_t = ω + (α + γI_{t-1})ε²_{t-1} + βσ²_{t-1}
I_{t-1} = 1 if ε_{t-1} < 0
```
**Citation:** Glosten, Jagannathan, Runkle (1993) J. Finance
**Edge:** Leverage effect

### Formula 341: TGARCH (Threshold)
```
σ_t = ω + αε_{t-1} + γε_{t-1}I_{t-1} + βσ_{t-1}
```
**Edge:** Asymmetric vol response

### Formula 342: FIGARCH (Long Memory)
```
σ²_t = ω/(1-β) + [1 - (1-βL)^{-1}(1-φL)(1-L)^d]ε²_t
```
**Citation:** Baillie, Bollerslev, Mikkelsen (1996) J. Econometrics
**Edge:** Fractional integration in variance

### Formula 343: APARCH
```
σ^δ_t = ω + α(|ε_{t-1}| - γε_{t-1})^δ + βσ^δ_{t-1}
```
**Citation:** Ding, Granger, Engle (1993)
**Edge:** Power transformation

### Formula 344: Realized GARCH
```
h_t = ω + βh_{t-1} + γRV_{t-1}
RV_t = ξ + φh_t + τ(z_t) + u_t
```
**Citation:** Hansen, Huang, Shek (2012) J. Applied Econometrics
**Edge:** Incorporates high-frequency data

### Formula 345: HAR-RV Model
```
RV_t = c + β_D RV_{t-1} + β_W RV_{t-1:t-5} + β_M RV_{t-1:t-22} + ε_t
```
**Citation:** Corsi (2009) J. Financial Econometrics
**Edge:** Heterogeneous autoregressive realized vol

### Formula 346: Multivariate GARCH (VECH)
```
vech(H_t) = C + A×vech(ε_{t-1}ε'_{t-1}) + B×vech(H_{t-1})
```
**Edge:** Full covariance dynamics

### Formula 347: Stochastic Volatility (Basic)
```
y_t = exp(h_t/2)ε_t
h_t = μ + φ(h_{t-1} - μ) + σ_η η_t
```
**Edge:** Latent volatility process

### Formula 348: Markov-Switching Variance
```
σ²_t = σ²_{s_t}, s_t ∈ {1,2,...,K}
```
**Edge:** Discrete volatility regimes

### Formula 349: Score-Driven (GAS) Model
```
f_{t+1} = ω + α s_t + β f_t
s_t = S_t × ∇_t
```
**Citation:** Creal, Koopman, Lucas (2013) J. Applied Econometrics
**Edge:** Observation-driven dynamics

### Formula 350: MIDAS (Mixed Data Sampling)
```
y_t = β_0 + β_1 Σ_{k=1}^{K} w_k(θ) x_{t-k/m} + ε_t
```
**Citation:** Ghysels, Santa-Clara, Valkanov (2004)
**Edge:** Mixed frequency regression

## SOURCE 3: Portfolio Theory Advanced (351-370)
**Citation:** Litterman (2003) "Modern Investment Management" Wiley
**Citation:** Grinold & Kahn (2000) "Active Portfolio Management" McGraw-Hill

### Formula 351: Information Ratio
```
IR = E[R_p - R_b] / σ(R_p - R_b) = α / ω
```
**Edge:** Active return per unit tracking error

### Formula 352: Information Coefficient
```
IC = Corr(forecast, realized)
```
**Edge:** Forecasting skill measure

### Formula 353: Breadth
```
BR = number of independent bets per year
```
**Edge:** Diversification of alpha

### Formula 354: Fundamental Law of Active Management
```
IR = IC × √BR
```
**Citation:** Grinold (1989) J. Portfolio Management
**Edge:** IR decomposition

### Formula 355: Transfer Coefficient
```
TC = Corr(optimal_weights, actual_weights)
```
**Edge:** Implementation efficiency

### Formula 356: Extended Fundamental Law
```
IR = IC × √BR × TC
```
**Edge:** With implementation constraints

### Formula 357: Alpha = IC × Volatility × Score
```
α_i = IC × σ_i × z_i
```
**Edge:** Expected alpha from signal

### Formula 358: Optimal Active Weight
```
w*_i = (α_i / σ²_ε,i) / Σ_j(α_j / σ²_ε,j)
```
**Edge:** Weight proportional to alpha/variance

### Formula 359: Risk Budget
```
RB_i = w_i × (∂σ_p/∂w_i) = w_i × β_i × σ_p
```
**Edge:** Risk contribution allocation

### Formula 360: Marginal Contribution to Risk
```
MCTR_i = ∂σ_p/∂w_i = (Σw)_i / σ_p
```
**Edge:** Risk sensitivity to weight change

### Formula 361: Maximum Diversification Portfolio
```
w* = argmax DR = (w'σ) / √(w'Σw)
```
**Citation:** Choueifaty & Coignard (2008)
**Edge:** Diversification ratio maximization

### Formula 362: Most Diversified Portfolio
```
w* = Σ^{-1} σ / (1'Σ^{-1}σ)
```
**Edge:** Closed-form max diversification

### Formula 363: Hierarchical Risk Parity
```
1. Cluster assets by correlation
2. Allocate recursively by inverse variance
```
**Citation:** López de Prado (2016) J. Portfolio Management
**Edge:** Hierarchical allocation

### Formula 364: Resampled Efficient Frontier
```
w̄ = (1/N) Σ_{n=1}^{N} w*_n(μ̂_n, Σ̂_n)
```
**Citation:** Michaud (1998)
**Edge:** Bootstrap-averaged weights

### Formula 365: Robust Optimization
```
min_w max_{μ∈U} w'μ - γw'Σw
```
**Edge:** Worst-case expected return

### Formula 366: CVaR Optimization
```
min_w CVaR_α(w) = min_{w,ζ} [ζ + (1/(1-α))E[max(-w'r - ζ, 0)]]
```
**Citation:** Rockafellar & Uryasev (2000) J. Risk
**Edge:** Tail risk minimization

### Formula 367: Mean-CVaR Portfolio
```
max_w E[w'r] - λ × CVaR_α(w)
```
**Edge:** Return vs tail risk tradeoff

### Formula 368: Omega Ratio
```
Ω(τ) = ∫_τ^∞ (1-F(r))dr / ∫_{-∞}^τ F(r)dr
```
**Citation:** Keating & Shadwick (2002)
**Edge:** Gain vs loss probability ratio

### Formula 369: Sortino Ratio (Full)
```
Sortino = (R_p - MAR) / √(∫_{-∞}^{MAR} (MAR-r)² f(r) dr)
```
**Edge:** Downside deviation normalization

### Formula 370: Calmar Ratio
```
Calmar = CAGR / |MaxDrawdown|
```
**Edge:** Return per unit drawdown risk

## SOURCE 4: Market Microstructure Extended (371-390)
**Citation:** Hasbrouck (2007) "Empirical Market Microstructure" Oxford
**Citation:** Harris (2003) "Trading and Exchanges" Oxford

### Formula 371: Glosten-Milgrom Spread
```
Ask = E[V|buy] = V + (μ/(μ+ε))Σ
Bid = E[V|sell] = V - (μ/(μ+ε))Σ
```
**Citation:** Glosten & Milgrom (1985) J. Financial Economics
**Edge:** Adverse selection spread

### Formula 372: Kyle Lambda (Continuous)
```
λ = σ_v / (2σ_u)
```
**Citation:** Kyle (1985) Econometrica
**Edge:** Price impact coefficient

### Formula 373: Easley-O'Hara PIN
```
PIN = αμ / (αμ + 2ε)
```
**Citation:** Easley & O'Hara (1992) J. Finance
**Edge:** Probability of informed trading

### Formula 374: VPIN Update
```
VPIN_n = Σ_{i=n-50}^{n} |V^B_i - V^S_i| / (50 × V_bucket)
```
**Citation:** Easley, López de Prado, O'Hara (2012) RFS
**Edge:** Volume-synchronized PIN

### Formula 375: Trade Classification (Lee-Ready)
```
Buy if: P > midquote or (P = mid and ΔP > 0)
```
**Citation:** Lee & Ready (1991) J. Finance
**Edge:** Trade direction inference

### Formula 376: Effective Spread (Detailed)
```
ES = 2 × D × (P - M)
D = +1 (buy), -1 (sell)
```
**Edge:** Actual trading cost

### Formula 377: Realized Spread
```
RS = 2 × D × (P - M_{t+Δ})
```
**Edge:** Profit to liquidity provider

### Formula 378: Price Impact Component
```
PI = 2 × D × (M_{t+Δ} - M_t) = ES - RS
```
**Edge:** Information component of spread

### Formula 379: Quoted Spread
```
QS = Ask - Bid
Relative_QS = (Ask - Bid) / Midpoint
```
**Edge:** Displayed liquidity cost

### Formula 380: Depth-Weighted Spread
```
DWS = Σ_i size_i × spread_i / Σ_i size_i
```
**Edge:** Size-adjusted spread

### Formula 381: Order Flow Toxicity
```
TOXICITY = |OFI| / Volume
```
**Edge:** Directional flow intensity

### Formula 382: Tick Test
```
Uptick: P_t > P_{t-1}
Downtick: P_t < P_{t-1}
Zero-tick: P_t = P_{t-1} (use last non-zero)
```
**Edge:** Alternative trade classification

### Formula 383: Information Share (Hasbrouck)
```
IS_i = [θ_i σ_η]² / Σ_j [θ_j σ_η]²
```
**Citation:** Hasbrouck (1995) J. Finance
**Edge:** Contribution to price discovery

### Formula 384: Component Share (Gonzalo-Granger)
```
CS_i = α_{⊥,i} / Σ_j α_{⊥,j}
```
**Citation:** Gonzalo & Granger (1995) J. Business & Econ Stats
**Edge:** Price discovery share

### Formula 385: Net Order Imbalance
```
NOI = (Buy_volume - Sell_volume) / Total_volume
```
**Edge:** Directional pressure

### Formula 386: Price Reversal Metric
```
Reversal = -Corr(R_t, R_{t-1})
```
**Edge:** Mean reversion indicator

### Formula 387: Weighted Price Contribution
```
WPC_i = (P_{i,close} - P_{i,open}) × Volume_i / Σ_j (P_{j,close} - P_{j,open}) × Volume_j
```
**Edge:** Venue contribution to price change

### Formula 388: Fragmentation Index
```
HHI = Σ_i s_i²
Fragmentation = 1 - HHI
```
**Edge:** Market concentration inverse

### Formula 389: Quote Stuffing Detection
```
QS_ratio = messages / trades
Anomaly if QS_ratio > threshold
```
**Edge:** Manipulation detection

### Formula 390: Layering Detection
```
Layering = Σ_levels (1 - execution_rate_level) × size_level
```
**Edge:** Spoofing indicator

## SOURCE 5: Commodities & Energy (391-410)
**Citation:** Geman (2005) "Commodities and Commodity Derivatives" Wiley
**Citation:** Pilipovic (2007) "Energy Risk" McGraw-Hill

### Formula 391: Convenience Yield
```
F_T = S × e^{(r - y + u)T}
y = convenience yield
u = storage cost
```
**Citation:** Brennan (1991) J. Finance
**Edge:** Benefit of holding physical

### Formula 392: Cost of Carry (Commodities)
```
F = S × e^{(r + u - y)T}
```
**Edge:** Futures pricing with storage

### Formula 393: Backwardation Condition
```
F < S (or F_{T2} < F_{T1} for T2 > T1)
y > r + u
```
**Edge:** Inverted term structure

### Formula 394: Contango Condition
```
F > S (or F_{T2} > F_{T1} for T2 > T1)
y < r + u
```
**Edge:** Normal term structure

### Formula 395: Calendar Spread
```
Spread = F_{T2} - F_{T1}
P&L = ΔSpread × Notional
```
**Edge:** Term structure trade

### Formula 396: Crack Spread (Refining)
```
Crack_3:2:1 = 2×Gasoline + 1×Heating_Oil - 3×Crude
```
**Edge:** Refining margin proxy

### Formula 397: Spark Spread (Power)
```
Spark = Power_price - Gas_price × Heat_rate
```
**Edge:** Power generation margin

### Formula 398: Dark Spread (Coal)
```
Dark = Power_price - Coal_price × Heat_rate - Carbon_cost
```
**Edge:** Coal power margin

### Formula 399: Crush Spread (Soybeans)
```
Crush = Soybean_meal + Soybean_oil - Soybeans
```
**Edge:** Processing margin

### Formula 400: Mean Reversion (Schwartz)
```
dS = κ(μ - ln S)S dt + σS dW
```
**Citation:** Schwartz (1997) J. Finance
**Edge:** One-factor commodity model

### Formula 401: Schwartz-Smith Two-Factor
```
ln S_t = χ_t + ξ_t
dχ_t = -κχ_t dt + σ_χ dW_χ (short-term)
dξ_t = μ_ξ dt + σ_ξ dW_ξ (long-term)
```
**Citation:** Schwartz & Smith (2000) Management Science
**Edge:** Short + long-term factors

### Formula 402: Gabillon Model
```
dS = κ(L - S)dt + σ_S S dW_S
dL = μ_L L dt + σ_L L dW_L
```
**Citation:** Gabillon (1991)
**Edge:** Spot and long-term price

### Formula 403: Gibson-Schwartz
```
dS = (r - y)S dt + σ_S S dW_S
dy = κ(ȳ - y)dt + σ_y dW_y
```
**Citation:** Gibson & Schwartz (1990) J. Finance
**Edge:** Stochastic convenience yield

### Formula 404: Electricity Spot Price (Spike)
```
ln P_t = f(t) + X_t + J_t
X_t = mean-reverting
J_t = jump process
```
**Edge:** Power price with spikes

### Formula 405: Weather Derivative (HDD/CDD)
```
HDD = max(65 - T_avg, 0)
CDD = max(T_avg - 65, 0)
```
**Edge:** Heating/cooling degree days

### Formula 406: Swing Option
```
V = max_{q_t ∈ [q_min, q_max]} E[Σ_t q_t × (P_t - K)]
subject to: Σ_t q_t ∈ [Q_min, Q_max]
```
**Edge:** Flexibility in take-or-pay

### Formula 407: Storage Valuation
```
V(S,I,t) = max[inject, withdraw, hold]
```
**Edge:** Real option on inventory

### Formula 408: Spread Option (Kirk)
```
C ≈ BSM(F_1 - F_2, K, σ_spread, r, T)
σ_spread² ≈ σ_1² + σ_2² × (F_2/(F_2+K))² - 2ρσ_1σ_2(F_2/(F_2+K))
```
**Citation:** Kirk (1995)
**Edge:** Spread option approximation

### Formula 409: Asian Option (Commodity)
```
Payoff = max(A - K, 0)
A = (1/n) Σ_i S_{t_i}
```
**Edge:** Average price option

### Formula 410: Quanto Commodity
```
F_quanto = F × e^{-ρ_{S,FX} × σ_S × σ_{FX} × T}
```
**Edge:** FX-adjusted commodity forward
