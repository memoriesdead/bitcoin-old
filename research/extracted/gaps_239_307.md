## SOURCE 1: Bond Mathematics (239-255)
**Citation:** Fabozzi (2007) "Fixed Income Mathematics" McGraw-Hill
**Citation:** Tuckman & Serrat (2011) "Fixed Income Securities" Wiley

### Formula 239: Bond Price (Discrete)
```
P = Σ_{t=1}^{n} C/(1+y)^t + F/(1+y)^n
```
**Variables:**
- C = coupon payment
- F = face value
- y = yield to maturity
- n = periods to maturity
**Edge:** Present value of cash flows

### Formula 240: Bond Price (Continuous)
```
P = Σ_{t=1}^{n} C×e^{-y×t} + F×e^{-y×T}
```
**Edge:** Continuous compounding version

### Formula 241: Macaulay Duration
```
D_Mac = [Σ_{t=1}^{n} t × CF_t/(1+y)^t] / P
```
**Citation:** Macaulay (1938) "Some Theoretical Problems"
**Edge:** Weighted average time to cash flows

### Formula 242: Modified Duration
```
D_Mod = D_Mac / (1 + y/k)
```
**Variables:**
- k = compounding frequency
**Edge:** Price sensitivity to yield change

### Formula 243: Dollar Duration (DV01)
```
DV01 = -∂P/∂y × 0.0001 ≈ D_Mod × P × 0.0001
```
**Edge:** Dollar change per basis point

### Formula 244: Convexity
```
Convexity = [Σ_{t=1}^{n} t(t+1) × CF_t/(1+y)^{t+2}] / P
```
**Citation:** Fabozzi (2007)
**Edge:** Second-order price sensitivity

### Formula 245: Price Change (Duration + Convexity)
```
ΔP/P ≈ -D_Mod × Δy + ½ × Convexity × (Δy)²
```
**Edge:** Better approximation for large yield changes

### Formula 246: Key Rate Duration
```
KRD_i = -∂P/∂y_i × (1/P)
```
**Citation:** Ho (1992) "Key Rate Durations"
**Edge:** Sensitivity to specific point on yield curve

### Formula 247: Effective Duration (OAS)
```
D_eff = (P_{-Δy} - P_{+Δy}) / (2 × P_0 × Δy)
```
**Edge:** Duration for bonds with embedded options

### Formula 248: Spread Duration
```
SD = -∂P/∂OAS × (1/P)
```
**Edge:** Sensitivity to option-adjusted spread

### Formula 249: Yield to Worst
```
YTW = min(YTM, YTC_1, YTC_2, ..., YTP)
```
**Variables:**
- YTC = yield to call
- YTP = yield to put
**Edge:** Most conservative yield assumption

### Formula 250: Current Yield
```
CY = Annual_Coupon / Current_Price
```
**Edge:** Simple income measure

### Formula 251: Z-Spread
```
P = Σ_{t=1}^{n} CF_t / (1 + r_t + Z)^t
```
**Variables:**
- r_t = spot rate for maturity t
- Z = Z-spread (constant spread)
**Edge:** Spread over entire spot curve

### Formula 252: I-Spread
```
I-Spread = YTM_bond - Swap_rate_{same_maturity}
```
**Edge:** Spread vs swap curve

### Formula 253: Asset Swap Spread
```
ASW = (Par - Dirty_Price)/PV01 + Fixed_coupon - Swap_rate
```
**Edge:** Spread in asset swap structure

### Formula 254: Repo Rate Implied
```
Repo_rate = (F - S) / S × (360/days)
```
**Variables:**
- F = futures price
- S = spot price
**Edge:** Implied financing rate

### Formula 255: Carry and Roll-Down
```
Carry = Coupon_income - Financing_cost
Roll = P(t+Δt, y_curve) - P(t, y_curve)
```
**Edge:** Return from holding position

## SOURCE 2: Interest Rate Derivatives (256-270)
**Citation:** Hull (2018) "Options, Futures, and Other Derivatives" Pearson
**Citation:** Brigo & Mercurio (2006) "Interest Rate Models" Springer

### Formula 256: Forward Rate Agreement (FRA)
```
FRA_payoff = N × (R_K - R_M) × τ / (1 + R_M × τ)
```
**Variables:**
- R_K = contracted rate
- R_M = market rate at settlement
- τ = accrual period
**Edge:** Cash-settled rate lock

### Formula 257: Interest Rate Swap Value
```
V_swap = B_fixed - B_float = Σ c_i × D(t_i) - (1 - D(T))
```
**Edge:** PV of fixed vs floating legs

### Formula 258: Swap Rate (Par)
```
S = (1 - D(T)) / Σ_{i=1}^{n} τ_i × D(t_i)
```
**Edge:** Rate making swap value = 0

### Formula 259: Basis Swap Spread
```
V = Σ (L_1 + s) × τ × D(t) - Σ L_2 × τ × D(t)
```
**Variables:**
- s = basis spread
- L_1, L_2 = different floating indices
**Edge:** Spread between floating rates

### Formula 260: Cross-Currency Basis
```
XCCY_basis = Forward_implied_rate - Actual_foreign_rate
```
**Edge:** FX-implied vs actual rate differential

### Formula 261: Cap/Floor Intrinsic
```
Intrinsic_cap = max(Forward - Strike, 0) × Notional × τ × D(T)
```
**Edge:** Minimum cap value

### Formula 262: Bermudan Swaption Value
```
V = max(Exercise_value, Continuation_value)
```
**Edge:** Early exercise optimal stopping

### Formula 263: Constant Maturity Swap (CMS)
```
CMS_rate ≈ Forward_swap + Convexity_adjustment
Conv_adj = ∂²S/∂F² × σ² × T / (∂S/∂F)
```
**Citation:** Hagan (2003) "Convexity Conundrums"
**Edge:** CMS vs forward swap rate

### Formula 264: In-Arrears Convexity
```
E[L_T × D(T)] = L_0 × D(T) × (1 + σ²Tτ/(1+L_0τ))
```
**Edge:** Adjustment for in-arrears payment

### Formula 265: Quanto Adjustment
```
Forward_quanto = Forward × exp(-ρ × σ_FX × σ_rate × T)
```
**Edge:** Cross-currency rate adjustment

### Formula 266: Timer Option
```
V = BSM(S, K, σ_timer, r, T_expected)
T_expected = V_budget / σ²_realized
```
**Edge:** Option with variance budget

### Formula 267: Range Accrual
```
Payoff = N × Σ_i c × 1{L_i ∈ [L_low, L_high]} / n
```
**Edge:** Coupon accrues only in range

### Formula 268: Inflation Swap
```
V = N × [(1 + fixed_rate)^T - CPI_T/CPI_0]
```
**Edge:** Fixed vs realized inflation

### Formula 269: TIPS Breakeven
```
Breakeven = Nominal_yield - TIPS_real_yield
```
**Edge:** Market-implied inflation

### Formula 270: Inflation Option (Year-on-Year)
```
YoY_floor = max(K - CPI_{t}/CPI_{t-1} + 1, 0)
```
**Edge:** Protection against low inflation

## SOURCE 3: Credit Derivatives Extended (271-285)
**Citation:** O'Kane (2008) "Modelling Single-name and Multi-name Credit Derivatives" Wiley
**Citation:** Schönbucher (2003) "Credit Derivatives Pricing Models" Wiley

### Formula 271: First-to-Default Basket
```
V_FTD = (1-R) × E[D(τ_1) × 1{τ_1 < T}]
τ_1 = min(τ_1, τ_2, ..., τ_n)
```
**Edge:** Protection on first default

### Formula 272: Nth-to-Default
```
V_NTD = (1-R) × E[D(τ_N) × 1{τ_N < T}]
```
**Edge:** Protection on Nth default

### Formula 273: CDO Tranche Premium
```
Spread_tranche = E[Protection_leg] / E[Premium_leg]
```
**Edge:** Fair spread for tranche

### Formula 274: CDO Tranche Loss
```
Loss_tranche = min(max(L - A, 0), D - A)
```
**Variables:**
- L = portfolio loss
- A = attachment point
- D = detachment point
**Edge:** Tranche absorbs losses in range

### Formula 275: Base Correlation
```
Find ρ: V_market(0, K) = V_model(0, K; ρ_base(K))
```
**Edge:** Correlation implied from equity tranche

### Formula 276: Gaussian Copula (CDO)
```
P(τ_i ≤ t | M) = N((N^{-1}(F_i(t)) - √ρ × M) / √(1-ρ))
```
**Citation:** Li (2000) "On Default Correlation"
**Edge:** Conditional default probability

### Formula 277: Large Pool Approximation
```
E[L] = ∫ LGD × N((N^{-1}(p) - √ρ×m)/√(1-ρ)) × φ(m) dm
```
**Edge:** Portfolio loss distribution

### Formula 278: Expected Tranche Loss
```
ETL = ∫_A^D (x-A) × f_L(x) dx + (D-A) × (1 - F_L(D))
```
**Edge:** Expected loss in tranche

### Formula 279: Delta (Credit)
```
Δ_credit = ∂V/∂S_CDS
```
**Edge:** Sensitivity to CDS spread

### Formula 280: CS01 (Credit Spread)
```
CS01 = -∂V/∂s × 0.0001
```
**Edge:** Value change per 1bp spread move

### Formula 281: Jump-to-Default (JTD)
```
JTD = V(pre-default) - V(post-default)
```
**Edge:** Loss if immediate default

### Formula 282: Recovery Lock
```
V = (R_contract - R_realized) × Notional × 1{default}
```
**Edge:** Trade on recovery rate

### Formula 283: Credit Swaption
```
V = E[max(V_CDS(T) - K, 0)]
```
**Edge:** Option on CDS spread

### Formula 284: Constant Maturity CDS
```
CMCDS = Forward_CDS + Convexity_adj
```
**Edge:** CDS indexed to constant maturity

### Formula 285: Credit Index Roll
```
Roll_cost = (On-the-run_spread - Off-the-run_spread) × Duration
```
**Edge:** Cost of rolling credit index position

## SOURCE 4: FX Derivatives (286-295)
**Citation:** Wystup (2017) "FX Options and Structured Products" Wiley
**Citation:** Clark (2011) "Foreign Exchange Option Pricing" Wiley

### Formula 286: Garman-Kohlhagen
```
C = S×e^{-r_f×T}×N(d_1) - K×e^{-r_d×T}×N(d_2)
d_1 = [ln(S/K) + (r_d - r_f + σ²/2)T] / (σ√T)
```
**Citation:** Garman & Kohlhagen (1983)
**Edge:** FX option pricing

### Formula 287: FX Delta Conventions
```
Δ_spot = e^{-r_f×T} × N(d_1)
Δ_forward = N(d_1)
Δ_premium_adj = Δ_spot - V/(S×N)
```
**Edge:** Different delta conventions in FX

### Formula 288: Risk Reversal
```
RR = σ(25Δ_call) - σ(25Δ_put)
```
**Edge:** Skewness of vol smile

### Formula 289: Butterfly
```
BF = [σ(25Δ_call) + σ(25Δ_put)]/2 - σ(ATM)
```
**Edge:** Curvature of vol smile

### Formula 290: ATM Conventions (FX)
```
ATM_DNS: Δ_call = -Δ_put (delta neutral straddle)
ATM_Forward: K = F
ATM_Spot: K = S
```
**Edge:** Different ATM strike conventions

### Formula 291: Vanna-Volga Adjustment
```
V_VV = V_BS + Δvanna × (σ - σ_ATM) + Δvolga × (σ - σ_ATM)²/2
```
**Citation:** Castagna & Mercurio (2007)
**Edge:** Smile adjustment via Greeks

### Formula 292: FX Barrier Rebate
```
Rebate_PV = R × P(barrier_hit) × E[D(τ_hit)]
```
**Edge:** Payment if barrier breached

### Formula 293: Window Barrier
```
V = V_barrier(t_start, t_end) vs V_barrier(0, T)
```
**Edge:** Barrier monitored in specific window

### Formula 294: Quanto FX Adjustment
```
F_quanto = F × exp(-ρ_{S,FX} × σ_S × σ_{FX} × T)
```
**Edge:** Correlation adjustment for quanto

### Formula 295: Dual Currency Deposit
```
Payoff = N × max(1 + r_d, (1 + r_f) × S_T/K)
```
**Edge:** Enhanced yield with FX risk

## SOURCE 5: Equity Derivatives Extended (296-307)
**Citation:** Gatheral (2006) "The Volatility Surface" Wiley
**Citation:** Taleb (1997) "Dynamic Hedging" Wiley

### Formula 296: Variance Swap Fair Strike
```
K_var = (2/T) × ∫_0^∞ [C(K)/K² + P(K)/K²] dK
```
**Citation:** Demeterfi et al. (1999)
**Edge:** Risk-neutral expected variance

### Formula 297: Variance Swap Payoff
```
Payoff = N_var × (σ²_realized - K_var)
```
**Edge:** Linear exposure to variance

### Formula 298: Volatility Swap Approximation
```
K_vol ≈ √K_var - (var(σ²))/(8×K_var^{3/2})
```
**Edge:** Convexity adjustment var→vol

### Formula 299: Corridor Variance Swap
```
K_corridor = ∫_L^U (2/K²) × [C(K) + P(K)] dK
```
**Edge:** Variance only when spot in range

### Formula 300: Gamma Swap
```
Payoff = Σ_i (S_i/S_0) × (ln(S_i/S_{i-1}))²
```
**Edge:** Gamma-weighted realized variance

### Formula 301: Dispersion Trade
```
P&L = σ²_index - Σ w_i² × σ²_i - 2×Σ_{i<j} w_i×w_j×ρ_{ij}×σ_i×σ_j
```
**Edge:** Index vol vs single stock vols

### Formula 302: Correlation Swap
```
Payoff = N × (ρ_realized - ρ_strike)
ρ_realized = [σ²_index - Σw_i²×σ²_i] / [2×Σ_{i<j}w_i×w_j×σ_i×σ_j]
```
**Edge:** Direct correlation exposure

### Formula 303: VIX Futures Fair Value
```
VIX_futures ≈ √(K_var_{30d,T}) × convexity_adj
```
**Edge:** VIX forward pricing

### Formula 304: VIX Option (Lognormal)
```
V = F×N(d_1) - K×N(d_2) with F = VIX_futures
```
**Edge:** Option on VIX futures

### Formula 305: Dividend Swap
```
V = PV(Div_implied) - PV(Div_strike)
```
**Edge:** Exposure to dividend stream

### Formula 306: Forward Starting Option
```
V = BSM(S, K×S_t/S_0, σ_forward, r, T-t)
```
**Edge:** Strike set at future date

### Formula 307: Cliquet (Locally Capped/Floored)
```
Payoff = Σ_i min(max(R_i, floor), cap)
R_i = S_{t_i}/S_{t_{i-1}} - 1
```
**Edge:** Sum of capped periodic returns
